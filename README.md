# RAG Syllabus Assistant 🎓

A production-grade Retrieval Augmented Generation (RAG) system 
built from scratch.

## What it does
Ask any question about a university syllabus and get 
intelligent AI-powered answers.

## Tech Stack
- LangChain — document loading and chunking
- ChromaDB — vector database
- Sentence Transformers — embeddings (all-MiniLM-L6-v2)
- Groq LLM (Llama 3.3) — answer generation
- Python — everything

## Phase 1 Complete ✅
- Document loading and chunking (700 chunks)
- Vector embeddings (384 dimensions)
- Semantic search
- LLM answer generation
